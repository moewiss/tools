## üîß Troubleshooting

### "No subtitle files downloaded"
**Causes:**
1. Video has no subtitles available
2. Selected language not available for that video
3. "Include Auto-Generated Subtitles" is not checked

**Solution:**
- Use one of the test URLs above
- Make sure "Include Auto-Generated Subtitles" is CHECKED
- Try selecting "English" (most common)

### "Subtitle file not loading in Windows Media Player"
**Causes:**
1. Subtitle file name doesn't match video file name
2. Files are not in the same folder

**Solution:**
1. Extract the ZIP file
2. Rename files to match:
   ```
   my_video.mp4
   my_video.srt  ‚Üê Must have exact same name
   ```
3. Put both files in the same folder
4. Open video in Windows Media Player
